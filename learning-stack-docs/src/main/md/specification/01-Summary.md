## 一 总体约定

### 标识约定
   1 英文大小写
   驼峰标识
   2 前缀
   所有项目均使用公司英文名称为开头, 且均为小写, 单词间使用'-'分隔, 如: bean-cloud-xxx
   3 后缀
   特定的模块用途, 使用特定的后缀, 如: 
   bean-cloud-xxx-core 核心业务, 
   bean-cloud-xxx-facade 接口等信息,
   bean-cloud-xxx-server 服务运行容器, 
   bean-cloud-xxx-spring-boot spring-boot相关
